# Bayesian Additive Modeling of Carbon Stock in Eucalypt Plantations

This repository contains the data and R scripts used in the article:

**Tomazela, B., Amorim, J. C., Silva, L. D., Guerra, S. P. S., Eufrade-Junior, H. J., & Prataviera, F. (20xx).**  
*Bayesian additive modeling of carbon stock in eucalypt plantations*.  
Journal of Sustainable Forestry.

---

## Overview

This repository provides code and data for modeling carbon stock in eucalypt plantations using Bayesian additive models. The approach allows flexible modeling of nonlinear relationships and full uncertainty quantification.

---

## Repository structure

- `data/` – dataset used in the analysis  
- `scripts/` – R scripts for preprocessing and modeling  
- `results/` – figures and outputs  

---

## Use of code and data

The code may be freely used for academic and research purposes.

The dataset is provided for reproducibility and must be used with proper attribution to the original sources.
